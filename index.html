02

Digital Image Compression with MATLAB GUI

This project is to develop a Graphical User Interface (GUI) application using MATLAB for digital image compression. Image compression is a technique used to reduce the size of an image file without significantly reducing its quality. The developed GUI allows users to load an image, and set the desired compression level. In addition, users can preview the compression result and compare it with the original image. This project is expected to provide a better understanding of image compression techniques that are useful for users in optimizing the storage and transmission of digital images using Matlab software.

Matlab, GUI
